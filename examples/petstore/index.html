Google Checkout Integration with Web 2.0 Pet Store Application

This project integrates Google Checkout APIs with the Web 2.0 Pet Store application from Sun Microsystem's Java BluePrints Program.

Instructions for building and running this application.

  1. Download glassfish and install it at a convenient location. We will refer to this location in the rest of the document as glassfish.home.
      The installation steps are:
    • java -Xmx256m -jar glassfish-installer-v1_ur1-p01-b02.jar
    • ant -f setup.xml
  2. Start glassfish and the derby database
    • cd glassfish
    • bin/asadmin start-database
    • bin/asadmin start-domain
  3. Download NetBeans 5.5 or later and install it at a convenient location. We will refer to this location as netbeans.home.
  4. Configure NetBeans to use glassfish. Here are the steps for this:
    • Start NetBeans, click on the Runtime tab, right-click on the Servers, and select "Add Server"
    • Use the default settings for most things, but change the Platform location to where you installed glassfish (glassfish.home).
    • For the admin username, use adminadmin as the password and select Finish.
  5. Checkout Java Petstore project by following these steps:
    • from a terminal, run cvs -d :pserver:guest@dev.java.net:/cvs co login
      Press enter when prompted for the password (blank password).
    • cd to the parent directory of google-checkout-java-sample-code/ and run cvs -d :pserver:gues@dev.java.net:/cvs co petstore
    • cd petstore/ws/bp-project/
    • Create a new file called javaeepasswords.txt and add the following contents to it
      AS_ADMIN_PASSWORD=adminadmin
    • cp build.properties.sample build.properties
    • Edit build.properties and change javaee.home to point to glassfish.home. Change javaee.server.passwordfile property to point to javaeepasswords.txt file.
      • cd petstore/ws/apps/petstore
      • ant setup
      • ant
    • From NetBeans, select File|Open Project and navigate to google-checkout-java-sample-code/examples/petstore and open that project
    • Right click on the project node and select run
    • If the compilation and deployment succeeds, the application will be launched in a browser. If not, go to http://localhost:8080/petstore to try the application.
