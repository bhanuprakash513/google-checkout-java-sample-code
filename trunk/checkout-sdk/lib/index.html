Using Google Checkout SDK

This SDK provides a library and instructions to integrate Google Checkout XML APIs in your Web applications. The remainder of the instructions assume that you unzipped the checkout sdk at checkout-sdk/ directory. Here are the steps to use this SDK:

  1. Copy checkout-sdk/checkout-sdk.jar to your WEB-INF/lib
  2. Edit your WEB-INF/web.xml and cut-and-paste the contents from checkout-sdk/web.xml
    1. If you need to add additional handlers to process checkout notifications, edit web.xml and change default value of the handlers to your classes.
    2. Use Checkout SDK Javadocs to extend notification handlers
  3. Copy over the contents of checkout-sdk/web to a suitable directory in your application. For example, web/checkout
  4. Your application is now ready to use Google Checkout APIs. Deploy it on tomcat or another application server to test the integration
